

#  Windows 12 (Visual)

A fan-made **Windows 12 desktop simulator** built using **Python + Pygame**.  
This project is a creative exploration of how a modern Windows UI could look and behave.

---

## 🚀 Features (Planned & In Progress)
-  Desktop environment
-  Taskbar
-  Start menu
-  Draggable windows
-  Fake apps (Notepad, Calculator)
-  Custom wallpapers & icons

---

##  Tech Stack
- Python 3
- Pygame

---

##  Project Structure
```
Windows-12-Visual/
│
├── main.py              # Entry point
├── settings.py          # Configs (resolution, colors, etc.)
│
├── core/                # Core engine logic
├── ui/                  # UI components (taskbar, desktop, etc.)
├── apps/                # Fake apps
├── assets/              # Images, fonts, icons
├── utils/               # Helper functions
│
├── .gitignore
└── README.md
```

---

## How to Run
1. Install dependencies:
   ```bash
   pip install pygame
   ```

2. Run the project:
   ```bash
   python main.py
   ```
